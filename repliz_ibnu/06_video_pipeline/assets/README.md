# Assets

Simpan asset reusable di sini.

Contoh subfolder yang disarankan:
- `backgrounds/`
- `icons/`
- `music/`
- `logos/`
- `overlays/`

Tahap awal scaffold ini belum mewajibkan asset eksternal karena template masih text-first.
Setelah workflow stabil, baru tambahkan icon, audio bed, atau visual pendukung per niche.
